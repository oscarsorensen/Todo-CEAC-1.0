En este ejercicio trabajé con DOM y aprendí a eliminar elementos de una página web utilizando JavaScript. Primero creé una tabla que contiene diferentes filas y luego apliqué la misma idea en un pequeño minijuego con un cuadrado en movimiento. Esto encaja bien con la idea de interactuar con los elementos de una página web de una manera más dinámica y realista. Creé las filas de la tabla utilizando JavaScript y añadí un evento para que cada fila se eliminara al hacer clic en ella. Más tarde, en el minijuego, hice lo mismo y añadí un evento al cuadrado móvil para que desapareciera al hacer clic en él. Solo utilicé conceptos de HTML y JavaScript de esta unidad, sin bibliotecas externas, y todo funciona correctamente en el navegador.
Esta actividad me ayudó a comprender mejor cómo manipular el DOM y reaccionar ante las interacciones de los usuarios. También muestra cómo se pueden utilizar estas habilidades en proyectos reales, por ejemplo, en juegos o páginas web interactivas, y refuerza los conceptos que aprendimos en clase de una manera práctica y sencilla.
