que tenemos un compilador instalado

Los programas en py, php, js se interpretan

En linux:
Simplemente escribid gcc

En mac:

instalador de XCode - Terminal Tools
(actually the same happend as what should happen in linux)

Descargar las herramientas de compilacion
https://sourceforge.net/projects/mingw/

compilar tanto en C como en C++
