# Ubuntu CLI commands

## System information
uname -a (viser alle systeminformationer)
hostnamectl (viser systemets hostname og relaterede oplysninger)
lscpu (viser CPU-arkitektur og detaljer)
timedatectl status (viser systemets dato og tid)

## System monitoring and management
top (viser kørende processer i realtid)
htop (interaktiv procesmonitor – kræver installation)
df -h (viser diskforbrug i læsbart format)
free -m (viser RAM-forbrug i MB)
kill <process id> (afslutter en proces)

## Running commands
<command> & (kører en kommando i baggrunden)
jobs (viser baggrundsprocesser)
fg <command number> (bringer en baggrundsproces til forgrunden)

## Service management
sudo systemctl start <service> (starter en service)
sudo systemctl stop <service> (stopper en service)
sudo systemctl status <service> (viser service-status)
sudo systemctl reload <service> (genindlæser service-konfiguration)
journalctl -f (viser logfiler i realtid)
journalctl -u <unit_name> (viser logs for en bestemt service)

## Cron jobs
crontab -e (redigerer planlagte opgaver)
crontab -l (viser planlagte opgaver)

## File management
ls (viser filer og mapper)
touch <filename> (opretter en tom fil)
cp <source> <destination> (kopierer filer)
mv <source> <destination> (flytter eller omdøber filer)
rm <filename> (sletter en fil)

## Directory navigation
pwd (viser nuværende mappe)
cd <directory> (skifter mappe)
mkdir <dirname> (opretter en mappe)

## File permissions and ownership
chmod [who][+/-][permissions] <file> (ændrer filrettigheder)
chmod u+x <file> (gør fil eksekverbar)
chown [user]:[group] <file> (ændrer ejer og gruppe)

## Searching and finding
find [directory] -name <search_pattern> (finder filer efter navn)
grep <search_pattern> <file> (søger tekst i filer)

## Archiving and compression
tar -czvf <name.tar.gz> [files] (komprimerer filer til arkiv)
tar -xvf <name.tar.[gz|bz|xz]> [destination] (udpakkker arkiv)

## Text editing and processing
nano <file> (åbner fil i nano-editor)
cat <file> (viser filindhold)
less <file> (viser fil sidevis)
head <file> (viser første linjer)
tail <file> (viser sidste linjer)
awk '{print}' <file> (printer hver linje i fil)
