Ejercicio: Gestión de clientes con opciones adicionales
Desarrolla un programa en Python que gestione una lista de clientes utilizando clases y estructuras de control. El programa debe permitir al usuario realizar las siguientes acciones:

Insertar un cliente: Pedirá al usuario los datos del cliente (nombre, apellidos, email) y lo añadirá a la lista.
Listar clientes: Mostrará todos los clientes con su ID correspondiente.
Actualizar un cliente: Pedirá el ID del cliente que desea modificar y los nuevos datos para actualizar.
Eliminar un cliente: Pedirá el ID del cliente que desea eliminar y pedirá confirmación antes de proceder.
Además, añade las siguientes opciones:

Cambiar a mayúsculas: Cambia todos los nombres y apellidos de los clientes a mayúsculas.
Cambiar a minúsculas: Cambia todos los nombres y apellidos de los clientes a minúsculas.
El programa debe guardar la lista de clientes en un archivo binario utilizando el módulo pickle para persistir los datos entre ejecuciones.

Restricciones
No se pueden usar estructuras de control complejas como bucles anidados o funciones recursivas.
Solo se puede utilizar el método pop() para eliminar elementos de la lista.
Las opciones adicionales (mayúsculas y minúsculas) deben ser implementadas como métodos en la clase Cliente.
Criterios de evaluación
Introducción y contextualización: El programa debe presentar una pantalla de bienvenida al inicio.
Desarrollo técnico correcto y preciso: El código debe seguir las mejores prácticas de programación, incluyendo el uso de clases y métodos adecuados.
Aplicación práctica con ejemplo claro: El programa debe funcionar correctamente y permitir la gestión de clientes de forma interactiva.
Cierre/Conclusión enlazando con la unidad: Al finalizar, el programa debe mostrar un mensaje indicando que ha terminado la ejecución.
