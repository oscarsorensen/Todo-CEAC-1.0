Contexto
En esta unidad, hemos aprendido cómo insertar datos en una base de datos utilizando Python y Tkinter. Hoy, vamos a combinar lo que hemos aprendido con un nuevo hobby: ir al gimnasio. Imagina que tienes una aplicación para registrar los entrenamientos realizados por tus amigos del gimnasio.

Enunciado paso a paso
Crea la interfaz gráfica:

Crea un marco (frame) en Tkinter.
Añade etiquetas y campos de entrada para registrar el nombre, apellidos y fecha del entrenamiento realizado por cada amigo del gimnasio.
Implementa la funcionalidad de inserción:

Escribir una función que se ejecute cuando se pulse un botón "Registrar".
Dentro de esta función, inserta los datos introducidos en la base de datos correspondiente.
Prueba tu aplicación:

Llena los campos y realiza el entrenamiento.
Verifica que los datos se guarden correctamente en la base de datos.
Restricciones
Solo puedes usar los conceptos y herramientas que hemos visto hasta ahora (Tkinter, MySQL Connector).
No utilices librerías externas adicionales.
Criterios de evaluación
Introducción y contextualización: 25%

El alumno debe entender el contexto del ejercicio y cómo combinarlo con su hobby.
Desarrollo técnico correcto y preciso: 25%

La interfaz gráfica debe estar correctamente implementada.
La función de inserción en la base de datos debe funcionar correctamente.
Aplicación práctica con ejemplo claro: 25%

El ejercicio debe ser completo y funcional.
El alumno debe probar su aplicación y verificar que los datos se insertan correctamente.
Cierre/Conclusión enlazando con la unidad: 25%

El alumno debe explicar cómo este ejercicio relaciona lo aprendido en la unidad con su hobby del gimnasio.
