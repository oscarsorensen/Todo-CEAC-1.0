# DAW Year 1 — CEAC Coursework

Full coursework from my first year of Desarrollo de Aplicaciones Web at CEAC, Valencia. 
Six modules covering the complete DAW curriculum.

This repo is not a portfolio — it's a working archive of everything built and learned 
during the course. Quality varies by intent: some folders are polished exercises, 
some are raw explorations, some are exam submissions.

---

## Modules

**1. Programación**
Python from fundamentals through OOP, GUI, file I/O, data structures, abstract classes, 
database access, server-side PHP, computer graphics, and a full AI unit covering 
vector databases, ChromaDB, and local LLM integration with Ollama.

**2. Bases de Datos**
SQL from first principles through joins, views, stored procedures, user privileges, 
ER modeling, and non-relational databases. Includes practical projects: login system, 
CRUD, admin panel, and canvas chart visualization.

**3. Lenguajes de Marcas**
HTML, CSS, JavaScript DOM manipulation, XML/XSD schema definition, XSLT, 
GitHub Pages publishing, and ERP integration. Ends with a browser-based video game.

**4. Proyecto Intermodular**
Business research and spin-off proposal, followed by practical exercises: 
SQL + Python connection, JSON microserver pattern, Jinja2 templates with Flask, 
form processing, and PHP + MySQL integration.

**5. Sistemas Informáticos**
Operating systems, networking fundamentals, and system administration basics.

**6. Entornos de Desarrollo**
Development environments, version control, and testing with pytest.
