UNIT 001 — Concept of a Class (Concepto de clase, clase, objeto, instancia)

What you must know

  • A class defines a new type with attributes and methods. (clase, propiedades, métodos)
  • An object (instance) is a concrete variable of that class. (objeto, instancia)
  • self refers to the current object inside methods.

Minimal pattern

class Cat:  # (clase)
    def __init__(self):           # (constructor, se verá en U005)
        self.name = ""            # (propiedad)
        self.age = 0              # (propiedad)

    def meow(self):               # (método)
        print("Meow, I'm", self.name)

# Create objects (instanciar objeto)
c1 = Cat()
c1.name = "Nina"
c1.age = 2
c1.meow()

Types of attributes

  • Instance attributes: defined on self. Each object has its own values. (propiedades de instancia)
  • Class attributes: defined on the class (rare in your course so far). (propiedades de clase)

Quick checks (exam)

  • Use clear names: name, price, email.
  • Show you can create objects and assign properties.
  • Add a tiny method that uses self.

Practice snippet

class Product:  # (Producto)
    def __init__(self):
        self.name = ""
        self.price = 0.0

    def info(self):
        return f"Product(name={{self.name}}, price={{self.price}})"

p = Product()
p.name, p.price = "Shaker", 9.99
print(p.info())
