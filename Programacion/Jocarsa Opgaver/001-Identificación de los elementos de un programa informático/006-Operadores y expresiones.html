Ejercicio: Incremento y decremento en edad En este ejercicio voy a utilizar var junto con los operadores += y -= para simular el paso del tiempo en la edad de un usuario. var es una palabra clave que nos permite declarar una variable cuyo valor puede cambiar. ```` var edad = 47; edad += 5; ```` Esto actualiza nuestra variable edad a 52. También utilizo console.log, para poder ver lo que hace el código en DevTools. ```` ```` Este ejercicio me ha permitido practicar cómo los operadores += y -= facilitan modificar el valor de una variable, en este caso la edad. En una aplicación real, como un gimnasio, sería útil para ajustar datos de los usuarios de forma sencilla y clara. Esto refuerza lo aprendido en la unidad sobre operadores aritméticos.
